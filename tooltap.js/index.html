A jQuery library to enable tooltips for elements.

What is it?

Tooltap enables small tooltips to show when the mouse is hovered over an element. This is useful when you need to present information to the user or give some pointers. For example, hover over the logo at the top of the page or hover over some of the greatest minds of our time:

Options + Syntax

There are several options to style or modify the tooltap to your liking. Here is an example of all the different options. Hover of each option for a description. Note: the value presented here are the default values of a tooltap.

$("element").tooltap({
write: "Description here",
id: "elementToolTap",
background: "#ffffff",
color: "#000000",
borderRadius: 4,
borderColor: "#c5c5c5",
shadow: "0px 2px 15px rgba(0,0,0,0.2)",
width: "auto",
maxWidth: 500
padding: 20,
triangleSize: 10
triangleOffset: 10,
speed: 250,
position: null
});
